Comandos básicos do terminal


CONCEITOS

•	símbolo “>” [é um redirecionador de fluxo]
•	Flags = [acrescentam, modificam, ou formatam a forma que esses comandos são devolvidos]
•	botão TAB [autocompleta o texto]
•	Qnd vc dá um comando e ñ aparece nada, deu certo (silence on success)
•	setinha p/ cima [navega por todos os últimos comandos executados]

COMANDOS
DESCRIÇÃO


•	cd (change directory) [caminho]  
cd / = Base do diretório C
cd Windows = entra na pasta Windows
cd .. = volta
cd workspace = entra na pasta workspace

•	dir (directory)  [lista de diretórios contidos na pasta em que estamos situados] 

•	mkdir (make directory) [ cria uma pasta]
mkdir workspace = cria uma pasta denominada “workspace”

•	del (delete) [deleta]
del workspace = deleta arquivos dentro da pasta workspace

•	rmdir (remove diretora) [deleta pasta]
rmdir workspace /S /Q]

•	echo [escreve]
echo hello > hello.txt  = cria um doc com o nome “hello”

•	cls (clear screen) [limpa o terminal]
